DS1620.py
=====================
A python port of cypherkey's RaspberryPi.Net DS1620 wrapper. Their wrapper can be found here:
https://github.com/cypherkey/RaspberryPi.Net

version
---------------------
0.0.1

usage
--------------------
```python
from DS1620 import DS1620

temp = DS1620(17, 18, 27)
x.get_temperature()
```
